I have created a v2 of the Maţřëullait cards. This deck only contains roots that are not referencing other roots for the use of their specification tables. This means that if a root is being referenced by other roots, that original root can act as a general form for all the roots being referenced.

As an example -LCW- meaning building fixture, yet a root like -BBY- (bathtub) references the -LCW- root for it's specification table, what I am proposing is that people learn -LCW- to refer to a bathtub without the -BBY- root by modifying -LCW- with VxCs affixes. So bathtub could be written as halcwaskell-ascal.
